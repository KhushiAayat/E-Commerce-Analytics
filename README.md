# E-Commerce-Analytics
Project overview: This project involves architecting an end-to-end analytics pipeline to tansform  raw e-commerce data into actionable insights. It demonstrates ptroficiency in cloud based data analysis and engineering.


Tech Stack: Cloud Data Warehouse- Google BigQuery, Language- SQL(Data Transformation and Modelling), Visualization( Power BI)

Project workflow
1) Data upload-  Uploaded raw Olist e-Commerce datasets ( customers, orders, order_items, products) on BigQuery
2) Transformation- Engineered a master data model by joining tables and calculating core KPIs , such as order  performance metrices.
3) Visualization-  Developed interactive dashboards in Power BI to visualize trends.

Key Achievements
1) Successfully implemented ETL(Extract-Transform-Load) workfllow
2) Utilized SQL joins  and data modeling techniques to  integrate multi-table relational data
3) Developed a scalable analytics aarchitecture using Google BgQuery
