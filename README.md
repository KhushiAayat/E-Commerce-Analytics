
Data Source:

Kaggle - Brazilian E-Commerce Public Dataset by Olist (https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)


Project Overview
This project demonstrates an end-to-end e-commerce analytics workflow, transforming raw transactional data into a structured analytical warehouse and delivering actionable business insights through an executive dashboard.

Using the Olist E-Commerce dataset, the project focuses on designing a scalable Star Schema in Google BigQuery, performing business-driven analytical queries, and developing an interactive Power BI dashboard for monitoring sales performance, customer behavior, and operational efficiency.

The objective was to move beyond basic drag-and-drop reporting by implementing robust data modeling, SQL-based transformations, and custom analytical measures that support reliable business decision-making.

Project Objectives
Transform raw e-commerce data into a structured analytical data model.
Design and implement a Star Schema for efficient reporting and scalability.
Analyze sales performance, customer payment behavior, and delivery operations.
Develop executive-level KPIs and business insights using SQL and DAX.
Create a professional, corporate-style Power BI dashboard for stakeholder reporting.
